# film-platform-navigation-analysis-report
该项目用于整理影视类网站平台导航分析报告信息，对首页导航层级划分、分类标签构成以及播放页信息区块安排进行系统归档。通过文字记录形成导航分析报告资料，便于对比不同影视平台在导航规划方面的差异。仓库仅保存公开页面信息。
